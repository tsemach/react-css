Example of Sudo Classes

Regular effect can be defined by:

a:link {
color: blue;
}

a:visited {
color: #8c8c9d;
}

a:link {
color: #F15A51;
}

However you can defined it with a class like: 

  .linkeffects:link {
    color: blue;
  }           
  .linkeffects:visited {
    color: #8c8c9d;
  }           
  .linkeffects:link {
    color: #F15A51;
  }
