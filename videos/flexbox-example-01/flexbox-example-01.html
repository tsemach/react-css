Example 03: Adding Matgins

Equal Hights

One of the greatest flex feature is equal hight

Adding 'align-items: flex-start' will cause the childs to start align but keep their natual height.
.call-outs-container {
   max-width: 1400px;
   display: flex;
   align-items: flex-start;
}

Adding 'align-items: flex-end' will cause the childs on the end align but keep their natual height.
.call-outs-container {
   max-width: 1400px;
   display: flex;
   align-items: flex-end;
}

Adding 'align-items: flex-center' will cause the childs to center align but keep their natual height.
.call-outs-container {
   max-width: 1400px;
   display: flex;
   align-items: flex-center;
}
